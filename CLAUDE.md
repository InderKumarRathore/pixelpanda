# Pixel Panda Portfolio Site

## Overview
Portfolio site for an iOS developer at pixelpanda.dev. Hosted on GitHub Pages.

## Tech Stack
- Plain HTML + CSS + vanilla JavaScript (no frameworks, no build tools)
- Font Awesome CDN for icons
- Google Fonts (Inter) for typography
- GitHub Pages hosting with CNAME: pixelpanda.dev

## Structure
- `index.html` — Single-page site with nav, hero, apps, contact, footer
- `moneymate/privacy-policy.html` — Privacy policy page
- `apps/moneymate/` — Moneymate app assets (icon.png, screenshots/)
- `apps/tradeok/` — TradeOK app assets (icon.png, screenshots/)
- `CNAME` — GitHub Pages custom domain

## Design
- Colors: #fafafa background, #1a1a1a text, #007AFF accent (iOS blue)
- Typography: Inter, generous whitespace
- Mobile-first responsive (breakpoints: 768px, 1024px)

## Email Obfuscation
Contact email is obfuscated to prevent crawlers. The scramble logic in the contact section must be preserved.
